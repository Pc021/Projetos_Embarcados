# Stock Car – Jogo em C para AT89C52

Este repositório contém o jogo *Stock Car*, desenvolvido como parte da disciplina de **Projetos de Sistemas Computacionais Embarcados** (1º semestre de 2025 da UFSCar).

## Conteúdo
- `stock_car.c`: Código-fonte em linguagem C
- `stock_car.hex`: Arquivo compilado para gravação no AT89C52
- `Stock_Car.pdf`: Relatório completo do projeto (desenvolvimento, hardware e software)

## Hardware
- Microcontrolador: AT89C52
- GLCD: AMPIRE 128x64
- Entradas: 2 botões (P1.0 e P1.1)

## Simulação
Simulação realizada no Proteus. O esquema pode ser visto no relatório.
